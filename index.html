A implementação de uma Assistente Virtual Modular

por

Leandro Pires de Souza

justificativa

Com os avanços tecnológicos e o surgimento de diversos conceitos como a Internet das Coisas (IoT) e de Wearable Devices, percebe-se uma convergência para o desenvolvimento de interfaces mais humanas e de uma interação mais natural com estes dispositivos. Além disso ainda não há uma proposta amplamente aceita para um padrão ou dispositivo que centralize todos estes conceitos. Uma assistente virtual modular teria a capacidade, tanto de promover uma interface mais natural e humana quanto de suprir a necessidade de uma tecnologia centralizadora para todos estes conceitos.

Objetivo

Implementar um Assistente virtual modular funcional e estabelecer protocolos, APIs, além de documentação que permita fácil desenvolvimento de módulos que servirão como expansão para a assistente e também promover a compatibilidade com padrões já existentes facilitando a interação entre módulos e dispositivos externos como dispositivos Wearable e dispositivos que utilizam o conceito de IoT.

Metodologia

  • A implementação se dará utilizando-se como hardware-base um Raspberry Pi 3 por ser um hardware barato, com considerável poder de processamento e com fácil adaptação para os mais diferentes tipos de projetos além de prover um grande número de interfaces externas.
  • A aplicação será baseada em Node JS e HTML5 devido a larga documentação disponível para estas linguagens, a existência de diversas bibliotecas que ampliam a capacidade de uma aplicação, além de uma ampla utilização no mercado (inclusive o mercado mobile) o que torna o desenvolvimento de módulos mais fácil.
